  1. case class ServiceIdentifier(name: String, path: Path, subdomain: String) extends Product with Serializable

    An identifier for Border Patrol to determine by path or by subdomain which service a request should be routed to

    An identifier for Border Patrol to determine by path or by subdomain which service a request should be routed to

    name

    The service name used for authentication

    path

    The external url path prefix that routes to the internal service

    subdomain

    A default fall-back when path is only /

    Example:

    1. Let's say we have a service named "enterprise", if we define the instance of that as:

      val ent = Service("enterprise", Path("/ent"), "default")
val biz = Service("business", Path("/biz"), "api")

      The following urls would match the services:

      • api.example.com/ent => ent
      • default.example.com/ => ent
      • default.example.com/biz => biz
      • api.example.com => biz
      • api.example.com/ent => ent These would not match the services:
      • a.api.example.com => ???
      • example.com => ???
